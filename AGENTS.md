# Static Pages Portfolio

## Dev Environment

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`

## Code Style

- Functional components
- Single quotes, no semicolons
- Tailwind CSS for styling
- Co-locate component-specific styles

## PR Instructions

- Title format: `[portfolio] Description`
- Run `npm run lint` before committing
- Test all landing pages before merge

## File Structure

```
static_pages_portifolio/
├── DESIGN.md
├── AGENTS.md
├── package.json
├── portfolio-hub/
├── landing-pages/
└── shared/
```
